# Atlas - Interactive Anatomy Illustration

## Project Overview
Interactive human anatomy web app using HTML, SVG, CSS, and jQuery. Body regions are clickable/hoverable SVG paths configured via `anatomy-config.js`.

## Key Files
- `index.html` — Main page with SVG body map
- `anatomy-config.js` — Region labels, URLs, and visibility settings
- `anatomy-script.js` — Interaction logic (hover, click behavior)
- `anatomy-style.css` — Styling and layout
- `modela.png` — Base anatomy image

## Notes
- Uses jQuery (included locally as `jquery.min.js`)
- No build step — static files served directly
